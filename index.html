Chapter 01

矢量分析与场论基础

电磁场理论的数学语言：从标量场到矢量场，从微分到积分定理。

1场的基本概念

场是物理量在空间中的分布。对于空间中每一点，都有一个确定的物理量与之对应。

标量场（温度、电位）只有大小；矢量场（电场、磁场）既有大小又有方向。
2矢量运算

点积结果为标量，叉积结果为矢量，方向遵循右手定则。

A·B = |A||B|cosθ  A×B = |A||B|sinθ n̂
3哈密顿算符 ∇

哈密顿算符是一个矢量微分算符，具有三种运算形式。

∇ = \frac{\partial}{\partial x}\,\hat{x} + \frac{\partial}{\partial y}\,\hat{y} + \frac{\partial}{\partial z}\,\hat{z}
∇Φ = 梯度（最大变化率方向）  ∇·A = 散度（源强度）  ∇×A = 旋度（旋转强度）
4梯度与方向导数

方向导数等于梯度在该方向上的投影，最大值为梯度的模。

\frac{\partial \Phi}{\partial l} = \nabla\Phi \cdot \hat{l} = |\nabla\Phi|\cos\theta
5重要定理
高斯散度定理
∮S A·dS = ∫V (∇·A) dV
面积分 → 体积分
斯托克斯旋度定理
∮C A·dl = ∫S (∇×A)·dS
线积分 → 面积分
✏ 例题 1.1 — 矢量点积与叉积

题目：已知矢量 A = 3\hat{x} + 2\hat{y} - \hat{z}，B = \hat{x} - \hat{y} + 2\hat{z}，求 A·B 和 A×B。

解题思路：点积用对应分量相乘再求和；叉积用行列式展开。

解答：

点积：A \cdot B = (3)(1) + (2)(-1) + (-1)(2) = 3 - 2 - 2 = -1

叉积：A \times B = \begin{vmatrix} \hat{x} & \hat{y} & \hat{z} \\ 3 & 2 & -1 \\ 1 & -1 & 2 \end{vmatrix} = 3\hat{x} - 7\hat{y} - 5\hat{z}

✏ 例题 1.2 — 散度计算

题目：求矢量场 \mathbf{F} = x^2\hat{x} + 2y\hat{y} + z\hat{z} 在点 (1,2,3) 处的散度。

解题思路：散度是对各分量分别求偏导再求和。

解答：

\nabla \cdot \mathbf{F} = \frac{\partial(x^2)}{\partial x} + \frac{\partial(2y)}{\partial y} + \frac{\partial(z)}{\partial z} = 2x + 2 + 1 = 2x + 3

在点 (1,2,3) 处：\nabla \cdot \mathbf{F} = 2(1) + 3 = 5

Chapter 02

静电场

由静止电荷产生的电场，具有无旋性（保守场）和有源性。

1电场强度与电位
电场强度
E = F/q₀
电位与电场关系
E = -∇Φ
负号表示电场强度指向电位降低最快的方向。
2真空静电场方程
高斯定律（微分形式）
\nabla \cdot E = \frac{\rho}{\varepsilon_0}
电荷密度 ρ 是电场的源
无旋性
∇×E = 0
静电场是保守场
\oint_S \mathbf{E} \cdot d\mathbf{S} = \frac{Q_{enclosed}}{\varepsilon_0} \quad \text{（积分形式）}
3电介质高斯定理

引入电位移矢量 D = ε₀E + P，消除束缚电荷的影响。

∇·D = ρfree  ∮S D·dS = Qfree
4边界条件
介质-介质界面
E1t = E2t
切向电场连续
法向分量
D2n - D1n = ρs
差值等于面电荷密度
5电场能量
能量密度
we = ½ε|E|²
总能量
We = ∫V we dV
✏ 例题 2.1 — 高斯定律求电场

题目：无限长均匀带电直线，线电荷密度为 τ，求距直线 r 处的电场强度。

解题思路：利用对称性，选圆柱面为高斯面，应用高斯定律积分形式。

解答：

由对称性，电场沿径向：\oint_S \mathbf{E} \cdot d\mathbf{S} = E \cdot 2\pi r L = \frac{\tau L}{\varepsilon_0}

解得：E = \frac{\tau}{2\pi\varepsilon_0 r}，方向沿径向向外。

✏ 例题 2.2 — 平行板电容器

题目：平行板电容器极板面积为 S，间距为 d，中间填充介电常数为 ε 的介质，求电容。

解题思路：设极板带电量 Q，用高斯定律求 D，再求 E，积分得电压 V，最后 C = Q/V。

解答：

D = \frac{Q}{S}, \quad E = \frac{D}{\varepsilon} = \frac{Q}{\varepsilon S}, \quad V = Ed = \frac{Qd}{\varepsilon S}

C = \frac{Q}{V} = \frac{\varepsilon S}{d}

Chapter 03

边值问题与求解方法

在给定边界条件下求解拉普拉斯方程或泊松方程，是电磁场的核心计算问题。

1边值问题类型
狄里赫利问题
给定边界电位值
Φ|boundary = f
诺依曼问题
给定边界法向导数
\frac{\partial \Phi}{\partial n} = g
混合边值问题
部分给电位，
部分给法向导数
2镜像法

用镜像电荷替代导体表面感应电荷，使边界条件自动满足。

关键条件：镜像电荷必须位于场区之外（导体内部），且满足边界条件。对于点电荷与接地导体平面，镜像电荷与原电荷等量异号。
典型应用：点电荷与无限大接地导体平面、点电荷与接地导体球、线电荷与导体平面。
3导体劈镜像法

对于夹角 α 的导体劈，仅当 α = π/n（n为正整数）时，才能用有限个镜像电荷满足边界条件。

原因：电荷的多次反射需要在有限步内闭合，劈角必须是 π 的整数分之一。
4分离变量法

系统化求解偏微分方程的经典方法：

① 选择坐标系 ② 分离方程 ③ 求通解 ④ 确定系数 ⑤ 叠加得解
适用于矩形、圆柱、球形等规则区域的静电场问题。
✏ 例题 3.1 — 镜像法

题目：点电荷 q 距接地导体平面为 d，求空间电位分布。

解题思路：在导体平面下方 d 处放置镜像电荷 -q，使平面电位为零。

解答：

镜像电荷：在 (0,0,-d) 处放置 -q

\Phi(x,y,z) = \frac{q}{4\pi\varepsilon_0}\left(\frac{1}{\sqrt{x^2+y^2+(z-d)^2}} - \frac{1}{\sqrt{x^2+y^2+(z+d)^2}}\right)

✏ 例题 3.2 — 镜像法验证

题目：验证例题 3.1 中导体平面 (z=0) 上电位为零。

解答：

令 z=0：\Phi(x,y,0) = \frac{q}{4\pi\varepsilon_0}\left(\frac{1}{\sqrt{x^2+y^2+d^2}} - \frac{1}{\sqrt{x^2+y^2+d^2}}\right) = 0 ✓

Chapter 04

恒定电流场

电荷做定向运动形成的场，具有无旋性和无源性（无源区）。

1恒定电流场方程
欧姆定律
J = σE
连续性方程
∇·J = 0
无旋性
∇×E = 0
2电流连续性原理

流入任意闭合曲面的电流等于流出的电流，这是电荷守恒定律在恒定电流条件下的体现。

∮S J·dS = 0
3边界条件
切向分量
E1t = E2t
切向电场连续
法向分量
J2n = J1n
恒定时法向电流密度连续
4功率损耗与静电比拟
功率损耗（焦耳热）
P = ∫V σ|E|² dV
单位体积：p = σ|E|²
静电比拟
\frac{G}{C} = \frac{\sigma}{\varepsilon}
电导与电容的类比
静电比拟：恒定电流场与无源区静电场具有相同数学形式，通过测量电容可间接测量电导。
5理想导电体

σ → ∞ 时，内部不能存在恒定电场（E = 0），否则 J = σE → ∞，物理上不可能。

✏ 例题 4.1 — 电流密度计算

题目：铜导线截面积为 1mm²，通过电流 10A，铜的电导率 σ = 5.8×10⁷ S/m，求电流密度和电场强度。

解题思路：J = I/A，再用欧姆定律 J = σE 求 E。

解答：

J = \frac{I}{A} = \frac{10}{1 \times 10^{-6}} = 10^7 \text{ A/m}^2

E = \frac{J}{\sigma} = \frac{10^7}{5.8 \times 10^7} \approx 0.172 \text{ V/m}

✏ 例题 4.2 — 功率损耗

题目：长 100m、截面积 1mm² 的铜导线，通过 10A 电流，求总功率损耗。

解题思路：先求电阻 R = L/(σA)，再用 P = I²R。

解答：

R = \frac{L}{\sigma A} = \frac{100}{5.8 \times 10^7 \times 10^{-6}} \approx 0.172 \ \Omega

P = I^2 R = 10^2 \times 0.172 = 17.2 \text{ W}

Chapter 05

麦克斯韦方程组

电磁场理论的核心方程，完整描述了电场与磁场的相互关系。

1麦克斯韦方程组概述

麦克斯韦方程组是电磁场理论的核心，完整描述了电磁场的基本规律，预言了电磁波的存在。

位移电流的引入是麦克斯韦最伟大的贡献之一，它使方程组在时变场中自洽。
2四个方程（微分形式）
第一方程 - 全电流定律
\nabla \times \mathbf{H} = \mathbf{J} + \frac{\partial \mathbf{D}}{\partial t}
传导电流和变化的电场都能产生磁场
第二方程 - 电磁感应定律
\nabla \times \mathbf{E} = -\frac{\partial \mathbf{B}}{\partial t}
变化的磁场能产生电场
第三方程 - 磁通连续性
∇·B = 0
磁场无源，磁力线总是闭合曲线
第四方程 - 高斯定律
∇·D = ρ
电荷以发散方式产生电场
3积分形式
\oint_C \mathbf{H} \cdot d\mathbf{l} = \int_S \mathbf{J} \cdot d\mathbf{S} + \int_S \frac{\partial \mathbf{D}}{\partial t} \cdot d\mathbf{S} \oint_C \mathbf{E} \cdot d\mathbf{l} = -\int_S \frac{\partial \mathbf{B}}{\partial t} \cdot d\mathbf{S} ∮S B·dS = 0 ∮S D·dS = ∫V ρ dV
4本构关系

麦克斯韦方程组需配合本构关系才能完整求解：

电介质：D = \varepsilon E
磁介质：B = \mu H
导体：J = \sigma E
5物理意义

变化的电场和变化的磁场彼此密切联系，相互激发，组成统一的电磁场整体。

在无源区（ρ = 0, J = 0），方程组预言了电磁波的存在。电磁波在真空中的传播速度等于光速，揭示了光的电磁本质。
✏ 例题 5.1 — 验证麦克斯韦方程组

题目：已知时谐电磁场 \mathbf{E} = E_0 \sin(kz - \omega t)\hat{x}，\mathbf{B} = B_0 \sin(kz - \omega t)\hat{y}，验证其满足真空中的麦克斯韦方程组。

解题思路：分别计算两个旋度方程和两个散度方程，验证等式成立。

解答：

① \nabla \cdot \mathbf{E} = \frac{\partial E_x}{\partial x} = 0 ✓（无源区 ρ=0）

② \nabla \cdot \mathbf{B} = \frac{\partial B_y}{\partial y} = 0 ✓

③ \nabla \times \mathbf{E} = -\frac{\partial E_x}{\partial z}\hat{y} = -kE_0\cos(kz-\omega t)\hat{y}

而 -\frac{\partial \mathbf{B}}{\partial t} = \omega B_0\cos(kz-\omega t)\hat{y}

需满足 kE_0 = \omega B_0，即 \frac{E_0}{B_0} = \frac{\omega}{k} = c ✓

Special Topic

麦克斯韦方程组 — 核心公式对照

微分形式描述局部性质，积分形式描述整体性质，两者通过高斯定理和旋度定理相互转换。

第一方程

推广的全电流定律
\nabla \times \mathbf{H} = \mathbf{J} + \frac{\partial \mathbf{D}}{\partial t}
\oint_C \mathbf{H} \cdot d\mathbf{l} = \int_S \mathbf{J} \cdot d\mathbf{S} + \int_S \frac{\partial \mathbf{D}}{\partial t} \cdot d\mathbf{S}
传导电流和变化的电场都能产生磁场。
磁场的旋度源包括传导电流密度 J 和位移电流密度 ∂D/∂t。

第二方程

推广的电磁感应定律
\nabla \times \mathbf{E} = -\frac{\partial \mathbf{B}}{\partial t}
\oint_C \mathbf{E} \cdot d\mathbf{l} = -\int_S \frac{\partial \mathbf{B}}{\partial t} \cdot d\mathbf{S}
变化的磁场能产生电场。
电场的旋度源是变化的磁感应强度。

第三方程

磁通连续性原理
∇·B = 0
∮S B·dS = 0
磁场是无源场，磁力线总是闭合曲线。不存在磁单极子。

第四方程

高斯定律
∇·D = ρ
∮S D·dS = ∫V ρ dV
电荷以发散的方式产生电场。
电位移矢量的散度等于自由电荷密度。
